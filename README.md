# Яндекс Минус — Инструкция по установке (Setup Guide)

Так как это проект на **Node.js** (JavaScript/React), аналогом `requirements.txt` из Python здесь выступает файл `package.json`. В нём уже прописаны все необходимые зависимости.

Чтобы развернуть проект у себя, каждому участнику команды нужно выполнить всего несколько шагов.

## Шаг 1. Установить Node.js
Если у кого-то из команды ещё не установлен Node.js, его нужно скачать и установить с официального сайта:
👉 [Скачать Node.js (выбирайте версию LTS)](https://nodejs.org/)

*Чтобы проверить, установлен ли Node.js, выполните в терминале команду `node -v`.*

## Шаг 2. Установка зависимостей
Откройте терминал (или командную строку), перейдите в папку проекта и запустите установку зависимостей. Это аналог `pip install -r requirements.txt`.

Команда:
```bash
npm install
```
*(Она скачает все нужные библиотеки в папку `node_modules`)*

## Шаг 3. Запуск проекта
После того как зависимости установлены, запускайте локальный сервер для разработки:
```bash
npm run dev
```

В терминале появится ссылка (обычно это `http://localhost:5173/`). Перейдите по ней в браузере, и вы увидите работающий Яндекс Минус!

---

## 🚀 Быстрый скрипт для самых ленивых (setup.sh)
Для удобства в корне проекта также создан исполняемый файл `setup.sh`. В Unix-системах (Mac, Linux) можно просто запустить его:
```bash
bash setup.sh
```
Он сам установит все зависимости и запустит сервер.
