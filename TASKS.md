# Яндекс Минус — Распределение задач по участникам

> **Запуск проекта:** `npm install && npm run dev`
> **Стек:** React 18 + Vite 5 + Zustand (стейт) + React Router (роутинг)
> **Общий стейт:** `src/store/gameStore.js` — **НЕ ТРОГАТЬ без согласования**

---

## Подготовка к работе

```bash
git clone <repo-url>
cd smehsmehsmeh
npm install
npm run dev  # откроется http://localhost:5173
```

Каждый работает **в своей ветке** (`git checkout -b feature/имя-задачи`), мёрж через PR.

## 👤 Участник 1 — Онбординг & Регистрация

**Зона ответственности:** Первое впечатление пользователя. Всё, что до главного экрана.

### Файлы:
- `src/pages/Registration.jsx` — основная страница (каркас готов)
- `src/pages/Registration.css` — стили (каркас готов)
- `src/components/ui/DifficultCaptcha.jsx` — капча (заглушка)
- `src/components/minigames/LicenseReader.jsx` — чтение лицензии вслух (заглушка)

### Задачи:
- [ ] **Капча из 2003 года** — `DifficultCaptcha.jsx`
  - Canvas с рандомным текстом (5-6 символов)
  - Искажение: rotate, skew, полосы, шум
  - 3 попытки, потом комментарий «Может, ты робот?»
  - Должно **выглядеть отвратительно** (это суть)

- [ ] **Выбор класса** — красивые карточки с анимацией hover
  - Маг / Воин / Лучник / Страж — классы уже в `gameStore.js`
  - Эффекты: свечение при выборе, тряска при наведении

- [ ] **Чтение лицензии вслух** — `LicenseReader.jsx`
  - `webkitSpeechRecognition` / `SpeechRecognition` API
  - Показать текст + подсветка распознанных слов в реальном времени
  - Порог: 70%+ совпадения → разблокировать «Принять»
  - Вызвать `gameStore.completeLicense()` при успехе (+100 ЯР)

- [ ] **Визуальная полировка** регистрации
  - Glitch-эффект на заголовке «ЯНДЕКС МИНУС»
  - Анимации переходов между шагами (fade / slide)
  - Страшилка-popup если юзер пытается закрыть страницу

### API контракт:
```js
// После регистрации вызвать:
import { useGameStore } from '../store/gameStore';
const register = useGameStore(s => s.register);
register({ userName, userClass: 'mage', monthlyIncome: 50000 });

// После лицензии:
const completeLicense = useGameStore(s => s.completeLicense);
completeLicense(); // +100 ЯР
```

- [ ] **d20 бросок** — визуальный 3D или 2D кубик

  - Нат.20 = бесплатная поездка (конфетти!)

- [ ] **Детектор устройства** — плашка на главном экране
  - `navigator.userAgent` → iPhone / Android
  - Анимированная плашка с текстом (из ТЗ)

- [ ] **Адрес-слайдеры** (ур.0-1) — доработать `Order.jsx`
  - Два слайдера: широта/долгота

## 👤 Участник 3 — Мини-игры & CV & Поддержка & Ожидание

**Зона ответственности:** Все мини-игры, CV-распознавание, поддержка, экран ожидания.

### Файлы:
- `src/pages/Waiting.jsx` — экран ожидания (каркас готов)
- `src/pages/Support.jsx` — поддержка (каркас готов)
- `src/components/minigames/SudokuModal.jsx` — судоку (заглушка)
- `src/components/minigames/GestureCapture.jsx` — CV-жест (заглушка)
- `src/components/minigames/PedalTapper.jsx` — педали (заглушка)
- `src/components/minigames/BalanceGame.jsx` — алкотестер (заглушка)

### Задачи:
- [ ] **Судоку для отмены** — `SudokuModal.jsx`
  - Генератор судоку (есть npm-пакеты: `sudoku` или написать свой)
  - Сетка 9×9 с навигацией по клеткам
  - Сложность зависит от `cancelCount` (см. ТЗ)
  - +30 ЯР за решение, −20 ЯР за сдачу
  - Вызвать: `gameStore.addRubles(30, 'Судоку решено')` / `addRubles(-20, 'Судоку сдался')`

- [ ] **Крути педали** — `PedalTapper.jsx`
  - Визуальный метроном: пульсирующий круг с целевым BPM (например, 120)
  - При тапе: проверить, попал ли ±200ms в бит
  - Показывать % попаданий
  - < 50% → анимация «водитель делает крюк»

- [ ] **CV-жест «67»** — `GestureCapture.jsx`
  - `navigator.mediaDevices.getUserMedia({ video: true })`
  - Подключить MediaPipe Hands или вашу готовую CV-модель
  - Распознать жест → вызвать `addRubles(67, 'Жест 67')` (разово)

- [ ] **Алкотестер** — `BalanceGame.jsx`
  - DeviceMotion API: `window.addEventListener('devicemotion', ...)`
  - Или fallback для десктопа: мышью двигать шарик в круг
  - 5 сек стабильности = прошёл

- [ ] **Поддержка** — полировка `Support.jsx`
  - Бот на ур.0-2: отвечает чушь с задержкой 2-5 сек
  - Бот на ур.2+: нормальные ответы (захардкоженные FAQ)
  - Ур.3: «живой чат» (но это всё ещё бот, просто с аватаркой)
  - Анимация «бот печатает...» (три точки)

- [ ] **Радар маньяка** — анимация на экране ожидания
  - Красная точка, движущаяся к пользователю
  - Зелёная точка (такси), которая должна доехать раньше
  - canvas или просто CSS-анимированные точки

### API контракт:
```js
import { useGameStore } from '../store/gameStore';
const addRubles = useGameStore(s => s.addRubles);
const incrementCancel = useGameStore(s => s.incrementCancel);

// Судоку решено:
addRubles(30, 'Судоку решено');
// Отмена:
incrementCancel();
addRubles(-30, 'Отмена поездки');
```

## Общие правила

1. **Стор (`gameStore.js`) — общий контракт.** Не меняйте его без согласования. Если нужно добавить поле — обсудите.
2. **CSS-переменные** уже настроены через `data-level` на `<html>`. Используйте `var(--accent)`, `var(--text-primary)` и т.д.
3. **Заглушки** уже стоят — заменяйте на реальные компоненты.
4. **Каждый компонент** имеет TODO-комментарий с именем ответственного.
5. **Коммитьте часто**, пишите осмысленные commit messages.
